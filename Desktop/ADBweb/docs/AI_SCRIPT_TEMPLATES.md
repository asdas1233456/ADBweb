## 🎯 模板系统概览

ADBweb 的脚本模板系统允许用户创建可重用的脚本模板，通过变量系统实现灵活的参数化配置。

### 核心特性

- **变量系统**：使用 `{{变量名}}` 语法
- **类型支持**：文本、数字、密码、选择等
- **验证机制**：必填验证、格式验证
- **分类管理**：按功能分类组织模板
- **使用统计**：跟踪模板使用频率

---

## 📝 模板语法

### 变量定义

```bash
# ADB 脚本示例
adb shell input tap {{x_coordinate}} {{y_coordinate}}
adb shell input text "{{username}}"
adb shell sleep {{wait_time}}
```

### 变量配置

```json
{
  "x_coordinate": {
    "type": "number",
    "description": "X坐标",
    "required": true,
    "default": "540"
  },
  "username": {
    "type": "text", 
    "description": "用户名",
    "required": true
  },
  "wait_time": {
    "type": "number",
    "description": "等待时间（秒）",
    "required": false,
    "default": "2"
  }
}
```

## 🔧 创建自定义模板

### 基本步骤

1. 进入 AI 脚本生成页面
2. 点击 "脚本模板" 标签
3. 点击 "创建模板" 按钮
4. 填写模板基本信息
5. 编写模板内容
6. 配置变量定义
7. 保存模板

### 最佳实践

- 使用清晰的变量命名
- 提供详细的变量描述
- 设置合理的默认值
- 添加必要的注释
- 遵循编码规范
